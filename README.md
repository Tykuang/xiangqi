# TERRY KUANG XiangQi

A modern Chinese chess (XiangQi) web application with AI opponent.

## Features

- 🎮 PVP and PVE modes
- 🧠 Four difficulty levels: Beginner, Advanced, Professional, Expert
- 📱 Responsive design
- 💾 Local storage for user authentication and game saves
- 🎨 Beautiful UI with smooth animations

## Getting Started

```bash
# Clone the repository
git clone https://github.com/Tykuang/xiangqi.git

# Navigate to the project
cd xiangqi

# Start the server (Python)
python serve.py

# Or use any HTTP server
python -m http.server 8000
```

## Dependencies

- No external dependencies required
- Built with vanilla HTML, CSS, and JavaScript
