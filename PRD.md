## 1. 项目背景与目标

### 1.1 一句话定位
一款**单页 Web 象棋游戏**,视觉语言采用 DSV 品牌 token 翻译版本,目标用户为需要轻量、无登录、可在浏览器中快速对弈的象棋爱好者与品牌展示场景。

### 1.2 目标
| 维度 | 目标 |
|---|---|
| **可玩性** | 完整中国象棋规则,支持人机(随机 AI)、人人本地对弈、双人远程(可选 V2) |
| **视觉** | DSV 品牌 token 化(深蓝主调 / 几何抽象 / 网格化 / 留白充足) |
| **可访问性** | WCAG AA 配色对比、键盘可达、移动端可玩 |
| **可交付** | 单页 HTML + CSS + 原生 JS,**无构建步骤**、双击 `index.html` 可跑 |
| **性能** | 首屏 < 100KB,加载 < 1s(Lighthouse Performance ≥ 90) |

### 1.3 非目标 (Non-Goals)
- ❌ 联网对战 / 账号系统 / 排行榜
- ❌ 残局库 / 开局库 / AI 引擎(初期只用随机合法着法)
- ❌ 棋谱导出 / 回放(V2 再做)
- ❌ 把 DSV logo / payoff 放在页面上(品牌风险)

## 5. 交互流程

### 5.1 走子主流程
```
[点击/拖拽己方棋子]
   ↓
[高亮所有合法着法] → 蓝点=可走 / 红圈=可吃
   ↓
[点击目标点 / 拖到目标点]
   ↓
[校验合法性] → 否:抖动提示 / 是:继续
   ↓
[执行走子 → 更新棋盘 → 写入历史 → 切换玩家]
   ↓
[检测将军/绝杀] → 是:弹胜负 / 否:若 AI 模式则触发 AI
```

### 5.2 AI 走子
```
[玩家落子成功]
   ↓
[若 AI 模式 且 轮到 AI]
   ↓
[枚举所有合法着法] → 随机选一
   ↓
[延时 0.5–3s(可调)] → 执行
```

### 5.3 错误与边界
| 场景 | 行为 |
|---|---|
| 拖到非法点 | 棋子弹回原位 + 红色短闪 |
| 点击对方棋子(无选子) | 提示「先选己方」 |
| 长按时误触 | 长按 200ms 才开始拖拽 |
| 移动端横屏过窄 | 棋盘缩小到 90vw,侧栏下移 |
| 同时多指触摸 | 取第一指,忽略其他 |

## 6. 数据结构(JS 内, 无后端)

```js
// 棋子编码: 1=帅 2=仕 3=相 4=车 5=马 6=炮 7=兵  (红方+)
//         -1=将 -2=士 -3=象 -4=车 -5=马 -6=砲 -7=卒 (黑方-)
board: number[10][9]   // 10 行 × 9 列, 0=空

state = {
  turn: 'red' | 'black',
  mode: 'pvp' | 'pve',
  history: Move[],          // {from:[r,c], to:[r,c], piece, captured}
  selected: [r, c] | null,
  legalMoves: [r, c][],
  inCheck: 'red' | 'black' | null,
  winner: 'red' | 'black' | 'draw' | null,
  undoStack: Move[]         // 最近 10 步
}
```

## 7. 规则引擎(走子合法性)

每种棋子的合法着法生成函数,纯函数返回 `Array<[r, c]>`:

| 棋子 | 算法要点 |
|---|---|
| **帅/将** | 九宫内每步 1 格,不能出九宫;对将规则(无子阻挡时不能直面) |
| **仕/士** | 九宫内斜走 1 格 |
| **相/象** | 走「田」字,不能过河,象眼被堵则不可走 |
| **车** | 直线任意距离,中间不能有子 |
| **马** | 走「日」字,蹩马腿方向不能有子 |
| **炮** | 直线走时中间无子;吃子时中间恰有 1 子 |
| **兵/卒** | 未过河只能前进 1;过河后可横走,不能后退 |

**自王规则**: 落子后己方将帅不能被对方任何棋子攻击,否则为非法。

## 11. 风险与对策

| 风险 | 对策 |
|---|---|
| 规则引擎 bug 导致走子错误 | 单元测试覆盖每种棋子的 5+ 场景 |
| 移动端拖拽事件冲突 | 用 pointer events 统一,长按 200ms 触发 |
| 品牌侵权风险 | 严格遵循 §11 翻译规则,不放 logo / payoff |
| AI 太弱玩家失望 | V1 文案说明「Random Move」,V2 引入简单估值 |
| 视障玩家无法使用 | aria-live 实时播报走子 + 胜负 |
